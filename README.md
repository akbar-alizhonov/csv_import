# Скрипт для поиска источников обращений

## Описание

Скрипт на Python с использованием SQL, который для каждого обращения (communication) находит последнюю предшествующую сессию пользователя на том же сайте и определяет рекламную кампанию, которая привела к обращению.

## Задача

Для каждого обращения необходимо:

1) Найти последнюю предшествующую сессию того же пользователя на том же сайте
2) Определить порядковый номер этой сессии для данного пользователя на данном сайте
3) Сохранить результат в CSV-файл


# Инструкция

## Технологии
Python 3.12
## Клонирование проекта
```
git clone https://github.com/akbar-alizhonov/csv_import.git
```
## Создание виртуального окружения
```
python3 -m venv venv 
```
## Активация виртуального окружения Mac/Linux
```
source venv/bin/activate
```
## Скопируйте и настройте значения переменных окружения
```
cp .env.example .env
```
## Установка зависимостей
```
pip3 install -r requirements.txt
``` 
## Запуск (в результате появится файл result.csv)
```
python3 main.py
```
